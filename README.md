Generate QR codes from text or URLs with custom colors and sizes. Scan QR codes from camera or uploaded images.

## Features

- Generate QR codes from text, URL, or any string
- Customize foreground/background colors and size
- Scan QR codes via camera or by uploading an image
- Download QR as PNG for print or web
- All processing in-browser; no server round-trip

## Why use this tool

Create QR codes for links, Wi-Fi configs, or small payloads without external sites. Scanning works for verification or digitizing printed codes.
